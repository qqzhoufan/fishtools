# AdGuard Home - DNS 广告过滤

端口: 3004 (Web UI), 53 (DNS)
默认账号: 首次访问配置向导中创建

访问地址: http://服务器IP:3004

## 功能说明
网络级广告和跟踪器拦截工具。
- 全网广告过滤（所有设备生效）
- 家长控制和安全浏览
- DNS-over-HTTPS / DNS-over-TLS 支持
- 详细的查询日志和统计

## 初始配置
1. 首次访问进入配置向导
2. 设置管理员账号和密码
3. 配置 DNS 监听端口（默认 53）
4. 完成后即可使用

## 客户端配置
将设备的 DNS 服务器设置为本服务器 IP 地址即可享受全网广告过滤。

## 注意事项
- DNS 端口 53 需要服务器防火墙放行
- 如与其他 DNS 服务冲突，请修改端口
