# Jellyfin - 媒体服务器

端口: 8096
默认账号: 首次访问自行创建

访问地址: http://服务器IP:8096

## 功能说明
开源家庭媒体系统，用于观看本地影音资源。
- 支持视频、音乐、照片管理
- 跨平台客户端（iOS、Android、智能电视等）
- 自动下载媒体元数据和海报
- 支持硬件转码加速

## 使用说明
1. 将影音文件放入 `./media` 目录
2. 首次访问创建管理员账号
3. 添加媒体库，指向 `/media` 目录
4. 等待扫描完成即可观看

## 性能优化
如需启用硬件加速，请编辑 `docker-compose.yaml` 取消 devices 部分的注释。
