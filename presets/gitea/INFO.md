# Gitea - 轻量 Git 服务

端口: 3003 (Web), 2222 (SSH)
默认账号: 首次访问完成初始化后注册

访问地址: http://服务器IP:3003

## 功能说明
轻量级自托管 Git 服务，类似 GitHub。
- 代码仓库管理
- Pull Request、Issue 跟踪
- Wiki、项目看板
- CI/CD 集成

## 初始配置
1. 首次访问会进入安装向导
2. 数据库配置已自动填充，直接下一步即可
3. 创建管理员账号
4. 配置服务器域名（可选）

## SSH 访问
Git SSH 克隆端口为 2222，使用方式：
```bash
git clone ssh://git@服务器IP:2222/用户名/仓库名.git
```

## 安全建议
生产环境请修改默认数据库密码。
