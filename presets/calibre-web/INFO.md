# Calibre-Web - 电子书管理

端口: 8083
默认账号: admin
默认密码: admin123

访问地址: http://服务器IP:8083

## 功能说明
基于 Web 的电子书管理和阅读平台。
- 在线阅读电子书（支持多种格式）
- 电子书元数据管理
- 书籍搜索和分类
- 支持发送到 Kindle

## 初始配置
1. 首次访问使用默认账号密码登录
2. 设置 Calibre 书库位置为：`/books`
3. 如果书库不存在，系统会自动创建
4. 登录后请立即修改默认密码

## 使用说明
将电子书文件（epub、mobi、pdf 等）放入 `./books` 目录即可在 Web 界面中管理和阅读。

## 安全建议
首次登录后请修改默认密码。
