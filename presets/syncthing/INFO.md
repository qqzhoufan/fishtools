# Syncthing - 文件同步

端口: 8384
默认账号: 无需登录

访问地址: http://服务器IP:8384

## 功能说明
开源的 P2P 文件同步工具，替代云盘同步。
- 多设备实时同步
- 端到端加密
- 无需中央服务器
- 版本控制和文件历史
- 选择性同步

## 使用说明
1. 访问 Web UI
2. 首次访问会提示设置用户名密码（可选）
3. 添加其他设备（通过设备 ID 配对）
4. 创建共享文件夹并选择同步的设备
5. 文件将自动在设备间同步

## 设备配对
在其他设备安装 Syncthing 后，通过设备 ID 互相添加即可建立同步连接。

## 数据目录
默认同步目录为 `./sync`，可在配置中添加更多文件夹。
