# Transmission - BT 下载客户端

端口: 9091
默认账号: admin
默认密码: admin123

访问地址: http://服务器IP:9091

## 功能说明
轻量级 BitTorrent 客户端，支持 Web 管理。
- 简洁易用的 Web 界面
- 支持磁力链接和种子文件
- 限速、调度等高级功能
- 远程访问和控制

## 使用说明
1. 访问 Web UI 并使用默认账号密码登录
2. 添加种子文件或磁力链接开始下载
3. 下载的文件保存在 `./downloads` 目录
4. 可将种子文件放入 `./watch` 目录自动开始下载

## 安全建议
首次登录后请修改默认密码。
